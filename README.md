
### Use bash

```bash
$ git clone https://github.com/ant-design/ant-design-pro.git --depth=1 -b v2
$ cd ant-design-pro
$ npm install
$ npm run dev:weapp         
```

### Use by docker

```bash
# open http://localhost

# yarn
$ yarn dev:weapp
$ yarn build:weapp

# npm script
$ npm run dev:weapp
$ npm run build:weapp

# production dev
$ npm run docker-prod:dev

# production build
$ npm run docker-prod:build
```
